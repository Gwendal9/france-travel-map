# Screenshots

Pour ajouter des captures d'écran :

1. Créer un dossier `screenshots/` à la racine
2. Ajouter vos images (PNG ou JPG)
3. Référencer dans le README

Suggestions de screenshots :
- Vue principale de la carte
- Modal d'un département
- Page statistiques
- Page planning
- Page wishlist
- Vue mobile

Format recommandé : 1920x1080 ou 1280x720
